# UserActivityMicroservice
Event-driven microservice built with Node.js, Kafka, and MongoDB for real-time user activity processing.

An **event-driven backend microservice** built with **Node.js, TypeScript, Kafka, and MongoDB** for real-time user activity processing.  

## Features

- Kafka **Producer** for publishing user activity events
- Kafka **Consumer** for asynchronous event processing
- MongoDB persistence with Mongoose
- Clean Architecture (Domain, Application, Infrastructure)
- REST API using Express
- Health check endpoint

 ## Tech Stack

- **Node.js**
- **TypeScript**
- **Express**
- **Kafka (kafkajs)**
- **MongoDB (Mongoose)**
